# 🛒 Amazon Product Review Analysis

## 📌 Project Overview
Analyzed Amazon Electronics customer reviews using Python and Data Science 
techniques. Performed sentiment analysis and data visualization on customer 
reviews and ratings.

## 🛠️ Technologies Used
| Tool | Purpose |
|------|---------|
| Python | Core programming |
| Pandas | Data cleaning & manipulation |
| NumPy | Statistical analysis |
| Matplotlib | Data visualization |
| Scikit-learn | ML sentiment classification |

## 📂 Dataset
- **Source:** Kaggle — Datafiniti Amazon Consumer Reviews
- **Link:** https://www.kaggle.com/datasets/datafiniti/consumer-reviews-of-amazon-products
- **Size:** 253MB (not included in repo due to GitHub size limit)
- **How to use:**
  1. Download dataset from the Kaggle link above
  2. Rename file to `amazon_electronics.csv`
  3. Place it in the project root folder
  4. Run `python amazon_review_analysis.py`

## 📊 Project Features
- ✅ Data cleaning and preprocessing of 30,000+ reviews
- ✅ Sentiment classification — Positive / Neutral / Negative
- ✅ TF-IDF Vectorization + Logistic Regression ML model
- ✅ 10 data visualizations saved to /visuals folder
- ✅ NumPy statistical analysis on ratings and review lengths
